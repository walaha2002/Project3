Predicting NCLEX Scores

This site attempts to predict NCLEX Pass/Fail scores using several different machine learning techniques. We fed the following inputs to the models:

>Cumulative GPA --testtesttest
>Maximum TEAS Score
>Number of TEAS Taken
>Failing Grade Count
>Previous Degree
>Time to Test
>Program Completion Time

We will use the following models and compare which model provides the highest prediction accuracy:

>Multiple Linear Regression Model
>Random Forest
>Extremely Randomized Trees
>Gradient Boosting Classification
